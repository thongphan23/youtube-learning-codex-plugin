# Brain2 And Vault Contract

Brain2 is required when the user asks to save durable learning for Thong Phan, compare against prior worldview notes, or create reusable content/strategy atoms.

## Speaker Profile

Speaker profiles may be drafted automatically when a video analysis is complete.

Target:

```text
01-Atomic/People/[person-slug].md
```

Required fields:

- Identity.
- Achievements.
- Distinctive lens.
- Representative quote if sourced.
- Worldviews.
- Learned sources.

## Worldview Library

Worldview atoms require user approval unless the user explicitly asked to save all.

Target:

```text
01-Atomic/Worldviews/worldview-[vietnamese-slug].md
```

Each worldview atom needs:

- YAML metadata.
- Core belief.
- Source person.
- Source video.
- Evidence quote or paraphrase.
- True cases.
- False cases.
- Content angles.
- At least 3 outgoing wiki links when writing into Brain2.

## Fallback

If Brain2 tools are unavailable, say so clearly. Prepare a `brain2-import.md` draft instead of claiming the note was saved.
